Array.js

Native array methods not supported by old browsers

Download this project as a .zip file Download this project as a tar.gz file

Welcome to array.js

array.js offers native array methods not supported by old browsers like:

isArray, indexOf, lastIndexOf, filter, forEach, map, some, every

How to use it

Just add 

<script type="text/javascript" src="array.js"></script>

and you'll be able to safely use native array methods

Documentation

You can find good documentation at mozilla developer page https://developer.mozilla.org/en-US/docs/JavaScript/Reference/Global_Objects/Array/prototype
