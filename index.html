Array.js

Array.js offers javascript array useful set methods like:

[1,2,3,4].subtract([2,3,5]); //return [1,4]
[1,2,3,4].intersect([2,3,5]); //return [2,3]
[1,2,3,4].exclusion([2,3,5]); //return [1,4,5]

and native array methods support for old browsers:

isArray, indexOf, lastIndexOf, filter, forEach, map, some, every

How to use it

Just add 

<script type="text/javascript" src="arraySet.js"></script>

Include this line if you want native array methods support (only needed for old browsers)

<script type="text/javascript" src="arrayMethodsSupport.js"></script>

and you'll be able to safely use set and native array methods

Documentation

Set methods

subtract:

a.subtract(b, fun)
Returns elements of array a not contained in array b. 
Uses fun(elemA, elemB) as comparision function. 
If it's not provided uses elemA == elemB

intersect:

a.intersect(b, fun)
Returns common elements between array a and array b. 
Uses fun(elemA, elemB) as comparision function. 
If it's not provided uses elemA == elemB

exclusion:

a.exclusion(b, fun)
Returns not common elements between array a and array b. 
Uses fun(elemA, elemB) as comparision function. 
If it's not provided uses elemA == elemB

Native array methods

You can find good documentation about native array methods at mozilla developer page or at ECMAScript page
